# Transent
A mini multicast socket file transfer software


## How to build
```
$ make clean && make
```
## How to run
Run program
```
# Run server
$ ./server <PORT>

# Run client
$ ./client <IP_SERVER> <PORT>
```

## For client
#### Program command
```
# Signup
SIGNUP user@password

# Login
LOGIN user@password

# Logout
LOGOUT user@password

# Find file
```
